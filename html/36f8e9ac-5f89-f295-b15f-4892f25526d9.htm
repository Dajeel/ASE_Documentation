      AppReset Class

      Represents a command to reset the canvas cursor to its initial position. Inherits from CanvasCommand and provides functionality to reset the drawing cursor to the origin point (0,0).

      Remarks

      This command takes no parameters and resets only the cursor position. Unlike the clear command, it does not affect any drawings on the canvas. The command is typically used to return to a known starting position for subsequent drawing operations.

      AppClear Class

      Represents a command to clear the canvas, removing all drawn elements and resetting it to its default state. Inherits from CanvasCommand and provides functionality to clear the entire canvas.

      Remarks

      This command takes no parameters and simply clears the entire canvas when executed. The canvas is reset to its default background color and all drawings are removed.
